# Coding Task PHP/JS

> Laravel&vuejs project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run migration
php artisan migrate

# launch dev server
php artisan serve
```
